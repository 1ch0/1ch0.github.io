1. 基础入门及变量使用

1.1 正则元字符

符号 描述 示例
. 任意一个字符 —
* 前面的字符重复任意次 x* xxxxx…
()* 字符串重复任意次 (shell)* shellshell…
+ 或 {1,} 前面字符重复一次以上 x+
? 前面字符重复0次或1次 x?
^ 行首 —
$ 行尾 —
< 或 \b 单词词首 —
> 或 \b 单词词尾 —
[] 字符中任意一个字符 [shell] -> s h e l l
[^.] 除了 . 之外的字符 —
[:alpha:] 关键字 —
| 或 a|bx, (a|b)xy
{n,} 最少多少次 —
{,n} 最多多少次 —
{m,n} m 次到 n 次 —
{m} 精确匹配多少次
(expr1)(expr2) \1 \2
& 前面表达式的结果

1.1.1 示例

ifconfig|egrep -o "\<(([0-9]|[1-9][0-9]|1[0-9][0-9]|2[0-4][0-9]|25[0-5])\.){3}([0-9]|[1-9][0-9]|1[0-9][0-9]|2[0-4][0-9]|25[0-5])\>"

1.4 变量

name=`hostname` # 将命令执行结果赋值给变量，但是结果只有一行，输出时 "$name" 保留格式

1.4.1 命名规则：

  • bash 不支持浮点数
  • 不要使用保留字
  • 不能以数字开头，只能数字、字母、下划线
  • 统一命名规则：驼峰命名法

1.4.2 变量的种类

  • 局部变量 在当前进程有效，子进程定义的变量不能传给父进程
  • 全局变量 父进程可以传给子进程。
  • 环境变量 export name=
echo $PID

echo $$

echo $PPID
pstree -p

1.4.3 变量引用

  • ${name} $name
  • “ “ : 弱引用，其中的变量引用会被替换为变量值

1.4.4 变量删除

  • 局部变量退出子进程
  • unset name

readonly name=test
declare -xs
# 查看常量

(name=test; echo $name)

2.2.2 位置变量

符号 意义
$# 参数个数
$? 前一条命令是否成功。
0 成功，1-255 失败，
若为脚本，判断脚本最后一条命令
exit 10，返回10
$* 所有参数，全部参数合为一个字符串
$@ 所有参数，每个参数为独立字符串
$0 命令本身，
加 `basename` 去除路径名
ln 软连接后显示软连接后文件名
$1 第一个字符串
${10} 第十个字符串
set – 清空位置参数
shift 丢弃第一个参数，参数整体左移一位 
#!/bin/bash
#
#***********************************************************************
#         Author:        1ch0
#         Date:          2022-01-04
#         FileName:      test.sh
#         Description:   1ch0 script
#         Copyright (C):2022 All rights reserved
#***********************************************************************
echo "1st arg is $1"
echo "2st arg is $2"
echo "name is `basename $0`"
echo "name is $@"
shift

echo "1st arg is $1"
echo "2st arg is $2"
echo "name is `basename $0`"
echo "name is $@"

echo {a..z}

5.2 sed

  • 用于文本内容做替换
sed '/user1/s/user1/u1/' /etc/passwd

  • 基本工作方式

    • 将文件以行为单位读取到内存（模式空间）
    • 使用 sed 的每个脚本对该行进行操作
    • 处理完成后输出该行

  • 替换命令 s

    • sed ‘s/lod/new/‘ filename
    • sed -e ‘s/lod/new/‘ -e ‘s/lod/new/‘ filename …
    • sed -i ‘s/lod/new/‘ ‘s/lod/new/‘ filename … 替换并写入

  • 带正则表达式的替换命令 s

    • sed ‘s/正则表达式/new/‘ filename
    • sed -r ‘s/扩展正则表达式/new/‘ filename
